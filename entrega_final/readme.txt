Este programa foi desenvolvido no formato Jupyter Notebook, com uso da linguagem Python.
É necessário a instalação das bibliotecas: pandas, numpy, matplotlib, seaborn, missingno e sklearn.

Para o iniciar, abra a consola no diretório dos ficheiros "VideoY.ipynb" e "video_games.csv" (na pasta Código), execute o comando 'jupyter notebook' e selecione o notebook.
O programa já deverá estar totalmente executado. Na necessidade de o exectuar novamente selecione a opção 'Run All', no menu 'Cell'.
